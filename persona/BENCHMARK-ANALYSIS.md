## ✍️ 글쓰기 스타일 분석

### 1. 문장 구조

#### 짧은 문장 사용
```
카스 프레시 아이스 맥주가 한정판으로 나왔길래,
어떤 차이가 있을까 궁금해서 먹어봤어요.

저는 평소에 테라 라이트를 주로 마셔요.
사기 전에 검색해 보고 결정하는 편인데,
후기를 보니 입에서 시원한 맛이 난다고 해서
도전해 봤어요.
```
- **특징**: 1~2줄 단위로 끊어서 작성
- **줄바꿈**: 문장마다 줄바꿈 (가독성 높음)
- **문단**: 3~5문장 단위로 빈 줄 삽입

#### 자연스러운 말투
- ~요체 기본 ("했어요", "같아요", "거든요")
- 구어체 표현 ("~인데,", "~길래", "~해서")
- 감탄/의문 섞기 ("왜 464ml로 했는지 싶어요")

### 2. 글 전개 패턴

#### 기본 구조 (제품 리뷰)
```
1. 도입: 왜 이 제품에 관심을 가졌는지
2. 구매 배경: 평소 습관 + 검색/비교 과정
3. 결론 선언: "결론은 호입니다" (먼저 말해버림)
4. 상세 설명: 구체적인 경험 묘사
5. 비유/비교: 익숙한 것에 비유 ("이클립스 시원함 정도를 약하게 해 놓은 느낌")
6. 장점 정리: 왜 좋았는지
7. 아쉬운 점: 솔직하게 (사소한 것)
8. 재구매 의사: 언급
9. 가격 정보: 명시
10. 추가 정보: 도움될 만한 것 (성분표 등)
```

#### 예시 분석
```
[도입] 카스 프레시 아이스 맥주가 한정판으로 나왔길래, 어떤 차이가 있을까 궁금해서 먹어봤어요.

[구매 배경] 저는 평소에 테라 라이트를 주로 마셔요. 사기 전에 검색해 보고 결정하는 편인데, 후기를 보니 입에서 시원한 맛이 난다고 해서 도전해 봤어요.

[결론 선언] 결론은 호입니다.

[상세 설명] 입에서 시원한 맛이 나요. 첫 맛부터 시원하지는 않고, 끝 맛에서 시원함이 살짝 느껴져요.

[비유] 이클립스 시원한 느낌처럼 나는데, 이클립스 시원함 정도를 약하게 해 놓은 느낌이 들어요. 아주 약간 정도로 나요.

[장점] 인위적이고 강하게 민트 느낌이 나지 않아서 좋았고, 부담 없이 입안이 깔끔하게 먹을 수 있는 맥주였어요.

[재구매 의사] 맥주의 청량함이 땡길 때는 재구매 의사 있어요.

[아쉬운 점] 아쉬운 점은 용량이 464ml예요. 500ml로 하지, 왜 464ml로 했는지 싶어요.

[가격] 가격은 2800원입니다!

[추가 정보] 맥주 살 때도 옆에 성분표 보시는 분들 있나요? 저는 새로운 맥주 마실 때는 항상 성분표를 봐요. 성분표 필요한 분들 있을 것 같아서 첨부합니다!
```

### 3. 제목 패턴

#### 유형별 제목 형식
```
[장소/카테고리] 제품명 키워드 후기
예: [을지로 힙한 술집] 음식이 맛있는 분위기 맛집 인쇄소에 다녀왔어요~

제품명 + 특징 + "후기"
예: 카스 프레시 아이스 입 안에서 시원한 맛이 난대서 마셔본 후기

장소 + 카테고리 + 상호명
예: 충무로역 국밥 맛집 / 진심

EP + 주제 + 부제
예: EP11. 30대 직장인 일상블로그│나의 소확행 여름 자취집밥 모음
```

#### SEO 키워드 배치
- 제목 앞부분에 메인 키워드
- 제목 길이: 25~45자
- 특수문자 활용: | / │ ~ ()

### 4. 톤 & 표현

#### 자주 쓰는 표현
```
도입부:
- "~가 궁금해서 먹어봤어요/써봤어요"
- "~길래 도전해 봤어요"
- "평소에 ~를 주로 ~해요"
- "검색해 보고 결정하는 편인데"

중간:
- "결론부터 말하면~"
- "~한 느낌이 들어요"
- "~같은 느낌이에요" (비유)
- "아주 약간 정도로 나요"

장점:
- "~해서 좋았고"
- "부담 없이 ~할 수 있었어요"
- "~라 만족해요"

아쉬움:
- "아쉬운 점은 ~예요"
- "왜 ~했는지 싶어요"
- "~면 더 좋았을 것 같아요"

마무리:
- "재구매 의사 있어요"
- "~할 때 추천해요"
- "가격은 ~입니다!"
- "필요한 분들 있을 것 같아서~"
```

#### 피하는 표현
```
❌ "강력 추천합니다!" (광고 느낌)
❌ "최고예요!" (과장)
❌ "꼭 사세요!" (푸시)
❌ "협찬받았습니다" → 글 마지막에 자연스럽게
❌ 딱딱한 설명체 ("본 제품은~")
❌ 과도한 이모지 남발
```

## 📸 이미지 사용 패턴

### 1. 이미지 수량
- 글당 평균 10~20장
- 제품 리뷰: 5~10장
- 맛집/여행: 15~25장

### 2. 이미지 종류
```
제품 리뷰:
1. 제품 전체 샷 (패키징)
2. 제품 상세 (라벨, 성분표)
3. 사용 장면 (손에 들고 있는 모습)
4. 결과물 (음식이면 완성 사진)
5. 디테일 컷

일상 글:
1. 상황 컷 (책상, 방, 창밖)
2. 음식 사진
3. 셀카 또는 거울샷
4. 소품/인테리어
```

### 3. 이미지 배치
- 텍스트 3~5문장마다 이미지 1장
- 이미지 → 텍스트 → 이미지 반복
- 중요 포인트에 이미지 배치

## ⚠️ 핵심: 자연스러움을 위한 랜덤화

### 1. 섹션 수/길이 랜덤화
```
❌ 틀린 방식:
- "섹션 10개로 작성"
- "각 섹션 4-6문장"
→ 기계적인 패턴 발생

✅ 올바른 방식:
- 섹션 수: 5~12개 사이 랜덤
- 문장 수: 2~8문장 랜덤
- 어떤 섹션은 짧게 (2~3문장)
- 어떤 섹션은 길게 (6~10문장)
- 가끔 한 문장짜리 섹션도 OK
```

### 2. 소제목/이모지 랜덤화
```
❌ 틀린 방식:
- 모든 섹션에 "이모지 + 소제목"

✅ 올바른 방식:
- 소제목 있는 섹션 60%
- 소제목 없이 바로 본문 40%
- 이모지 쓰는 섹션 / 안 쓰는 섹션 섞기
- 구분선(---) 가끔 사용
```

### 3. 어조/톤 변화
```
기본 톤에서 이런 변화 섞기:

혼잣말:
- "아 근데 이게~"
- "음... 어떻게 말해야 하지"
- "이거 은근 괜찮더라고요"

질문:
- "이거 써보신 분 있나요?"
- "저만 그런가요?"
- "다들 어떻게 생각하세요?"

감탄:
- "진짜 대박"
- "와 이건 좀..."
- "완전 만족!"

망설임:
- "솔직히 좀 그랬어요"
- "글쎄요... 호불호 있을 것 같아요"
- "아직 잘 모르겠어요"

잡생각:
- "근데 요즘 이런 거 많이 나오더라고요"
- "이거 살 때 다른 것도 봤는데"
- "아 그리고 하나 더"
```

### 4. 이미지 배치 랜덤화
```
❌ 틀린 방식:
이미지 → 텍스트 → 이미지 → 텍스트 (규칙적 반복)

✅ 올바른 방식:
[이미지 3장 연속]
짧은 텍스트 2~3문장

[텍스트 길게 6~8문장]
[이미지 1장]

[이미지 2장]
중간 텍스트

[텍스트만 - 이미지 없이]

[이미지 4~5장 몰아서]
짧은 설명
```

**이미지 그룹핑 규칙:**
- 1~4장씩 그룹으로 묶기
- 텍스트-이미지 비율 랜덤
- 이미지 없는 섹션도 포함 (20~30%)
- 이미지 연속 배치 후 설명하는 패턴
